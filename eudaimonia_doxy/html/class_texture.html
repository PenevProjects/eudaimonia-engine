Detailed Description

Controls texture objects. Needed because textures need to be glDestroyed.

Can only be created through default constructor, because OpenGL encapsulates glTexture objects within the API. Memory management for glTextures is not controlled by the user - this makes copy and copy assignment constructors dangerous to use, so they are deleted.
